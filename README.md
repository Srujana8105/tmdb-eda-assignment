# TMDB EDA Assignment

This project demonstrates how to fetch movie data using the TMDB API and perform basic exploratory data analysis (EDA) using Python and Pandas.

## Steps Performed

1. Fetched movie data from TMDB API
2. Converted API response into a Pandas DataFrame
3. Stored data for analysis
4. Performed basic EDA:
   - Displayed first 5 rows
   - Generated summary statistics
   - Checked missing values
   - Counted movies by genre

## Technologies Used

- Python
- Pandas
- Requests
- TMDB API

## Note

API key is not included in this project for security reasons.
